タイムライン可視化 DOTファイル（Graphviz）記述仕様書（実績準拠・完全版）
1. 全体・環境設定 (Graph)

    文字コード: 記述不具合（文字化けなど）を防ぐため、冒頭で必ず charset = "UTF-8"; を宣言する。

    配置エンジン: 座標固定配置を正確に反映するため、layout = neato; を使用する。

    重なり・分散防止: ノードが被らないよう overlap = false; および sep = "+4"; を指定する。

    背景色: 視認性を考慮し、白（bgcolor = "#ffffff";）で統一する。

    エッジ補正: 線がノードの下をくぐらないよう splines = true; を有効化する。

2. ノード（登場人物）の共通・デザイン仕様

    形状・塗りつぶし: ボックス型（shape = box;）とし、style = "filled"; を指定する。

    サイズ: width = 1.4; height = 0.3; fixedsize = true; に固定する。

    フォント: fontname = "sans-serif"; をデフォルトとする。

    枠線の非表示化（重要）: 枠線を目立たなくするため、デフォルト設定で fillcolor = "#fcfcfc"; color = "#fcfcfc"; penwidth = 0.1; とし、背景色と枠線色を同化させる。

    文字色（fontcolor）による勢力・属性別の色分け:

        大覚寺統・討幕過激派・挙兵当事者: fontcolor = "#cc0000"; （鮮やかな赤。後醍醐天皇、護良親王、文観、楠木正成など）

        持明院統（皇族・院）: fontcolor = "#0000cc"; （鮮やかな青。後伏見院、花園院、光厳天皇など）

        鎌倉幕府（得宗・内管領・探題）: fontcolor = "#004400"; （深い緑。北条高時、長崎円喜、北条仲時など）

        主要な実務派公卿・主要武士（中立または動向注視）: fontcolor = "#000000"; （黒。万里小路宣房、足利尊氏、佐々木道誉など）

        諸国の有力武士・周辺人物・文化宗教: fontcolor = "#444444"; （濃いグレー。土岐、島津、大内、夢窓疎石など）

3. 時期に応じた「名前（名乗り）」の制御規則

歴史上の人物の改名（名乗り変更）に対応するため、DOTファイルの時期（年月）に応じて表示を厳密に制御する。

    改名する前までの時期: label には当時の古い名前を記載する。（1324年時点では「足利高氏」、1331年時点でもまだ「足利高氏」のまま）

    改名するまさにそのタイミング（月）: label には 新しい名前（古い名前） の形式で併記する。

    改名した後の時期: label には新しい名前のみを記載する。

4. ハイパーリンク（Wikipedia連携）の規則

    生の日本語URLの採用: URLエンコード（パーセントエンコード）は一切使用しない。ブラウザやツールでの視認性を保つため、生の日本語文字列のURLをそのまま記述する。

    ターゲット指定: リンクをクリックした際に必ず別タブで開く設定（target="_blank";）をノードごとに付与する。

5. エッジ（関係線）の重要度・配色別仕様

関係性の重みに応じて、線の太さ（penwidth）、矢印の大きさ（arrowsize）、および配色を使い分ける。グラフの視認性を最優先するため、label によるテキスト表示は一切行わず、すべての説明文は tooltip に集約する。

    【重要度：太】中枢の支配・計画検挙・最高指令

        仕様: penwidth = 4.0; arrowsize = 1.4;

    【重要度：中】通常の君臣関係、明確な同心や対立、直接の追討軍指揮

        仕様: penwidth = 3.2; arrowsize = 1.2;

    【重要度：細】補助的・裏の関係、姻戚、遠因、水面下の連携

        仕様: penwidth = 2.5; arrowsize = 1.0;

    エッジの配色規則:

        幕府・実務系・統制・追討関係: color = "#005500"; （幕府の緑系）または color = "#444444";

        討幕・挙兵・同志の連帯関係: color = "#cc0000"; または color = "#ff4444";

        持明院統・宮廷警戒関係: color = "#0000aa";

        婚姻・一門・穏やかな連携・遠隔の遠因: color = "#555555"; または color = "#666666";

6. 時系列に沿ったツールチップ（tooltip）の文言維持規則

    文言の完全一致: 連続する時代（前後の月や期）のDOTファイルにおいて、あるノードやエッジの状況・関係性の内容自体に変化がない場合は、tooltip 内の文章や表現を一切変更せず、完全に同一の文字列を維持して引き継ぐ。
